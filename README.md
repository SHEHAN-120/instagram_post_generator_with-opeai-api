# 🧠 Instagram Social Media Post Generator

This project uses the OpenAI API to generate Instagram-style social media posts based on our prompt. It is designed to help users quickly create effective contents.

## ✨ Features

* Automatically generates short-form posts using OpenAI's GPT model
* Analyzes effectiveness of the generated post for Instagram
* Restructures content for better audience engagement
* Uses `.env` to keep your OpenAI API key private

## 🚀 Technologies Used

* Python
* OpenAI API
* Jupyter Notebook (`.ipynb`)
* `dotenv` for secure API key loading
* Markdown for formatted output
* VS Code for development
